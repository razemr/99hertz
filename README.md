## Getting Started

### Installation

1. Open `api` folder
2. Install server NPM packages
```
npm i
``` 
5. open `ui` folder
6. Install client NPM packages
```
npm i
```

### Execution
1. Server
```
npm run start:dev
```
2. Client
```
npm run start
```
